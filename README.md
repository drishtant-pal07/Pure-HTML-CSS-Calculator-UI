🔢 Pure HTML CSS Calculator UI
A modern, responsive calculator interface built entirely with HTML5 and CSS3. This project demonstrates advanced frontend styling, layout management, and UI component design without the use of any JavaScript.

🎨 Design Highlights
Flexbox/Grid Layout: Used to perfectly align the number pad and operator buttons.

Hover Effects: Interactive button animations that change color or depth when hovered.

Responsive UI: The layout adapts to different screen sizes (Mobile & Desktop).

Custom Typography: Focused on readability and clean numerical display.

🛠️ Technical Skills Shown
CSS Component Architecture: Organizing styles for buttons, display screens, and containers.

Pseudo-classes: Using :hover and :active to simulate a "clickable" feel.

Box-Shadow Mastery: Creating depth (3D look) for the calculator buttons.
